= Astra API (ACS) kubernetes API endpoints de recursos
:allow-uri-read: 


La API de recursos de la API de Kubernetes proporciona acceso de solo lectura a los recursos de la API de Kubernetes para filtrar los que son compatibles con Astra en un clúster determinado.
